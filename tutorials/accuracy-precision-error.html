Accuracy vs Precision vs Error

Core definitions

Accuracy and precision are two distinct properties of a measurement, and confusing them is one of the most common conceptual errors in experimental science. They are related but independent — a measurement set can be precise without being accurate, accurate without being precise, both, or neither.

TermDefinitionAssociated error type
AccuracyHow close a measurement is to the true or accepted valueSystematic error
PrecisionHow reproducible a set of measurements is — how close repeated measurements are to each otherRandom error
TruenessISO term for closeness to the true value (essentially synonymous with accuracy)Bias / systematic error
ReproducibilityPrecision achieved across different days, operators, or instrumentsCombined error sources
RepeatabilityPrecision achieved by the same operator, same instrument, same dayShort-term random error
The archery target analogy

Imagine shooting arrows at a target where the bullseye is the true value. Accurate but not precise: arrows scattered around the bullseye — average is correct but spread is large. Precise but not accurate: arrows clustered tightly together but far from the bullseye — reproducible but systematically wrong. Accurate and precise: arrows tightly clustered on the bullseye — the ideal result. Neither: arrows scattered randomly with no pattern — the worst outcome.

Types of error

Every measurement contains error. This is not a failure — it is an inherent property of measurement. The goal is to understand, quantify, and minimise error, and to report it honestly alongside your results. Errors fall into two fundamental categories.

Error typeCharacteristicEffect on resultsCan averaging eliminate it?
Systematic errorConsistent, directional offset — always shifts results the same wayReduces accuracy; causes biasNo — averaging does not remove a consistent offset
Random errorUnpredictable fluctuations in both directionsReduces precision; causes scatterYes — averaging multiple measurements reduces random error
Gross error (blunder)Large, obvious mistake — misreading, transcription errorProduces an outlier far from all other valuesIdentified and excluded, not averaged

Systematic error

Systematic errors are consistent biases that shift every measurement in the same direction by a similar amount. Because they are consistent, they do not cancel out when you average replicate measurements — they are present in the mean. Identifying and eliminating systematic errors is the primary goal of instrument calibration and method validation.

Common sources of systematic error

SourceExampleCorrection
Instrument miscalibrationBalance reading 0.05 g high on every measurementCalibrate with certified reference weights before use
Reagent impurityNaOH solution absorbed atmospheric CO₂, reducing actual concentrationStandardise against a primary standard; use freshly prepared solutions
Matrix interferenceA coloured sample absorbing light at the measurement wavelengthPrepare a sample-matched blank; use a different analytical method
Environmental conditionsTemperature-dependent volume measurement from a non-temperature-corrected buretteWork at, or correct for, the calibration temperature (typically 20°C)
Analyst techniqueConsistently reading the top of the meniscus instead of the bottomTraining and technique standardisation
ContaminationTrace metal contamination in reagents shifting all readings highUse analytical-grade reagents; acid-wash glassware; use process blanks

Random error

Random errors cause measurements to scatter unpredictably above and below the true value. They arise from uncontrollable fluctuations — slight differences in reading a scale, minor temperature variations, vibrations, or electronic noise. Random errors follow a statistical distribution and can be characterised and reduced by taking multiple measurements.

Common sources of random error in the laboratory

  • Reading a scale or digital display — the last digit of any instrument is inherently uncertain.
  • Slightly variable technique when pipetting — minor differences in plunger speed affect dispensed volume.
  • Environmental fluctuations — air currents near a balance, temperature changes during a long run.
  • Electronic noise in detector systems — particularly at low signal levels.
  • Sample variability — biological samples are inherently variable.

Quantifying error — absolute and relative error

Error can be expressed as an absolute quantity or as a fraction of the true or measured value. Both are useful in different contexts.

Error calculations
Absolute error = |Measured value − True value|

Relative error = (Absolute error / True value) × 100%

Example: True value = 25.00 mL; Measured = 24.85 mL
Absolute error = |24.85 − 25.00| = 0.15 mL
Relative error = (0.15 / 25.00) × 100% = 0.60%
Relative error (%) is often more informative — 0.15 mL is negligible in 1 L but significant in 1 mL.

Mean, standard deviation, and relative standard deviation

When multiple measurements are taken, their statistical properties describe both the precision of the method and the reliability of the result.

Key statistical descriptors
Mean (x̄) = Σxᵢ / n

Standard Deviation (SD) = √[ Σ(xᵢ − x̄)² / (n − 1) ]

Relative Standard Deviation (RSD or %CV) = (SD / x̄) × 100%

Example: Readings: 10.2, 10.4, 10.1, 10.3, 10.2 mg/mL
Mean = 10.24 mg/mL
SD = 0.114 mg/mL
RSD = (0.114 / 10.24) × 100% = 1.11%
RSD < 2% is generally acceptable for routine analytical work. RSD < 5% is often the threshold for method validation.
Why n − 1 in the standard deviation formula?

Using (n − 1) in the denominator rather than n gives the sample standard deviation, which is an unbiased estimate of the population standard deviation. Using n would systematically underestimate variability when working with small sample sets (as is almost always the case in a laboratory). This correction is called Bessel's correction. Most calculators and spreadsheet functions (STDEV in Excel) use n − 1 by default.

Error propagation

When a calculated result depends on multiple measured quantities, each with its own uncertainty, the uncertainties combine — they "propagate" into the final result. Understanding this prevents the false confidence of reporting a highly precise-looking result that is actually very uncertain.

Simplified propagation rules
For addition / subtraction (A ± B):
Absolute uncertainty of result = √(δA² + δB²)

For multiplication / division (A × B or A / B):
Relative uncertainty of result = √[(δA/A)² + (δB/B)²]

Example: C = A × B where A = 5.00 ± 0.05, B = 2.00 ± 0.02
Rel. uncertainty = √[(0.05/5.00)² + (0.02/2.00)²] = √(0.01² + 0.01²) = 0.0141 = 1.41%
Result: C = 10.00 ± 0.14

Improving measurement quality

Knowing the sources of error gives you a direct path to reducing them. The following systematic approach addresses both accuracy and precision.

GoalStrategy
Improve accuracyCalibrate instruments against certified reference standards; use blanks and controls; standardise reagents; validate the method against reference materials
Improve precisionTake multiple replicates (n ≥ 3, ideally ≥ 5); use automated dispensing where possible; maintain consistent technique; control environmental conditions
Detect systematic errorInclude a positive control with a known true value; analyse a certified reference material; compare results with an independent method
Identify gross errors (blunders)Apply outlier tests (Grubbs test, Dixon Q test); never discard an outlier without documented justification; cross-check transcribed values
Report uncertainty correctlyAlways report mean ± SD (or ± SEM where appropriate); state n; report RSD; never imply more precision than the instrument provides
Key Takeaways

What to remember from this tutorial

  • Accuracy is closeness to the true value; precision is reproducibility. They are independent properties — you can have one without the other.
  • Systematic error causes consistent bias and cannot be eliminated by averaging. It must be identified and corrected at its source.
  • Random error causes scatter and is reduced by taking multiple measurements and averaging.
  • Absolute error = |measured − true|; relative error (%) = (absolute error / true value) × 100%.
  • Report results as mean ± SD with n stated. Calculate RSD to assess precision: <2% is generally acceptable.
  • Use n − 1 (not n) in the standard deviation formula for sample data — this is Bessel's correction.
  • Errors propagate through calculations — a precise-looking final number may carry large uncertainty if input measurements were uncertain.
